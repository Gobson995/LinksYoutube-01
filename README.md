# 🔗 Links YouTube

Página web desenvolvida para centralizar e compartilhar meus vídeos do YouTube de forma simples e organizada.

O projeto foi construído com o apoio da **Rocketseat**, com foco em prática de desenvolvimento front-end e criação de páginas pessoais.

---

## 🚀 Sobre o projeto

Este projeto funciona como um **hub de links**, permitindo reunir em um único lugar os principais conteúdos do meu canal no YouTube.

A proposta é oferecer uma navegação rápida, limpa e acessível para quem deseja acompanhar meus vídeos.
